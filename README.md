## 🚀 تشغيل الموقع | Getting Started

### الطريقة 1: مباشرة
فتح ملف `index.html` في أي متصفح حديث

### الطريقة 2: Live Server (VS Code)
```bash
# تثبيت امتداد Live Server في VS Code
# ثم الضغط بزر الماوس الأيمن على index.html
# واختيار "Open with Live Server"
```

### الطريقة 3: GitHub Pages
1. اذهب إلى **Settings** في المستودع
2. اختر **Pages** من القائمة الجانبية
3. اختر **Branch: main** ثم اضغط **Save**
4. سيكون موقعك على: `https://username.github.io/alrahma-dental-clinic`
